# Баг-репорт: Краш при вводе спецсимволов в поиск

**ID:** BUG-001  
**Название:** Приложение крашится при вводе спецсимволов в строку поиска  
**Шаги воспроизведения:**
1. Открыть приложение Bluesky.
2. Перейти в поиск.
3. Ввести `!@#$%^&*()_+`.
4. Нажать "Поиск".
